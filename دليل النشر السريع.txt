═══════════════════════════════════════════════════════════════════════
📱 الطريقة 1: النشر على Streamlit Cloud (مجاني - موصى به)
═══════════════════════════════════════════════════════════════════════

⏱️ **الوقت المتوقع:** 10-15 دقيقة

**الخطوات:**

1️⃣ **إنشاء حساب GitHub:**
   ✓ اذهب إلى: https://github.com/signup
   ✓ أنشئ حساب مجاني (استخدم بريدك الإلكتروني)

2️⃣ **إنشاء Repository جديد:**
   ✓ بعد تسجيل الدخول، اضغط "New repository"
   ✓ اسم Repository: health-data-analyzer
   ✓ اختر: Public
   ✓ اضغط "Create repository"

3️⃣ **رفع الملفات إلى GitHub:**
   ✓ اضغط "uploading an existing file"
   ✓ ارفع هذه الملفات:
     - app.py
     - requirements.txt
     - template.pptx
     - README.md
   ✓ ارفع مجلد .streamlit أيضاً
   ✓ اضغط "Commit changes"

4️⃣ **إنشاء حساب Streamlit Cloud:**
   ✓ اذهب إلى: https://share.streamlit.io/signup
   ✓ اضغط "Continue with GitHub"
   ✓ وافق على الصلاحيات

5️⃣ **نشر التطبيق:**
   ✓ اضغط "New app"
   ✓ Repository: اختر health-data-analyzer
   ✓ Branch: main
   ✓ Main file path: app.py
   ✓ اضغط "Deploy!"

6️⃣ **انتظر (2-3 دقائق):**
   ✓ سيتم تثبيت المكتبات تلقائياً
   ✓ سيفتح التطبيق تلقائياً

7️⃣ **احصل على الرابط:**
   ✓ الرابط سيكون مثل:
     https://health-data-analyzer-xxxxx.streamlit.app
   ✓ انسخ هذا الرابط وشاركه مع أي شخص!

═══════════════════════════════════════════════════════════════════════
💻 الطريقة 2: التشغيل المحلي (على جهازك فقط)
═══════════════════════════════════════════════════════════════════════

**للاختبار السريع على جهازك:**

1️⃣ تثبيت المكتبات:
   ```
   pip install -r requirements.txt
   ```

2️⃣ تشغيل التطبيق:
   ```
   streamlit run app.py
   ```

3️⃣ فتح المتصفح:
   ✓ سيفتح تلقائياً على: http://localhost:8501
   ✓ أو افتح الرابط يدوياً

⚠️ **ملاحظة:** هذا للاختبار فقط - لن يعمل من أجهزة أخرى

═══════════════════════════════════════════════════════════════════════
📱 الاستخدام على الجوال
═══════════════════════════════════════════════════════════════════════

**Android:**
1. افتح Chrome
2. اذهب إلى رابط التطبيق
3. استخدمه مثل أي موقع

**iPhone/iPad:**
1. افتح Safari
2. اذهب إلى رابط التطبيق
3. اضغط زر المشاركة (مربع مع سهم)
4. اختر "Add to Home Screen"
5. الآن أصبح لديك أيقونة على الشاشة الرئيسية!

🎯 استخدام التطبيق (سهل جداً!)
═══════════════════════════════════════════════════════════════════════

1️⃣ **ارفع ملفات Excel:**
   ✓ اضغط "Browse files"
   ✓ اختر الملفات
   ✓ اضغط "تحميل وتحليل البيانات"

2️⃣ **اختر الوحدة:**
   ✓ من القائمة المنسدلة

3️⃣ **املأ البيانات (اختياري):**
   ✓ عدد السكان، القرى، المسؤولين

4️⃣ **أنشئ التقرير:**
   ✓ اضغط "إنشاء تقرير PowerPoint"
   ✓ حمّل الملف
