## 🌟 المميزات

✅ **يعمل على أي جهاز:**
- 💻 الكمبيوتر (Windows, Mac, Linux)
- 📱 الجوال (Android, iOS)
- 🖥️ التابلت

✅ **سهل الاستخدام:**
- واجهة عربية بسيطة
- بدون تحميل أو تثبيت

✅ **نفس الوظائف:**
- رفع ملفات Excel
- اختيار الوحدة
- إدخال البيانات الإضافية
- تحميل التقرير PowerPoint

---

## 🚀 طريقة التشغيل المحلي

### 1. تثبيت المتطلبات:
```bash
pip install -r requirements.txt
```

### 2. تشغيل التطبيق:
```bash
streamlit run app.py
```

### 3. فتح المتصفح:
- سيفتح التطبيق تلقائياً على: `http://localhost:8501`

## ☁️ النشر المجاني على Streamlit Cloud

### الخطوات:

#### 1️⃣ إنشاء حساب GitHub:
- اذهب إلى: https://github.com
- أنشئ حساب مجاني

#### 2️⃣ رفع الملفات:
- أنشئ repository جديد
- ارفع جميع الملفات:
  - `app.py`
  - `requirements.txt`
  - `template.pptx`
  - `.streamlit/config.toml`

#### 3️⃣ إنشاء حساب Streamlit Cloud:
- اذهب إلى: https://streamlit.io/cloud
- سجّل دخول بحساب GitHub

#### 4️⃣ نشر التطبيق:
- اضغط "New app"
- اختر repository
- اختر ملف `app.py`
- اضغط "Deploy"

#### 5️⃣ احصل على الرابط:
- سيكون الرابط مثل: `https://your-app.streamlit.app`
- شارك هذا الرابط مع أي شخص!

## 🎯 كيفية الاستخدام

### 1. رفع الملفات:
- اضغط "Browse files"
- اختر ملف Excel واحد أو أكثر
- اضغط "تحميل وتحليل البيانات"

### 2. اختيار الوحدة:
- من القائمة المنسدلة
- اختر وحدة محددة أو "المحلة ثان (كاملة)"

### 3. إدخال البيانات (اختياري):
- عدد السكان
- القرى والعزب
- أسماء المسؤولين

### 4. إنشاء التقرير:
- اضغط "إنشاء تقرير PowerPoint"
- انتظر قليلاً
- اضغط "تحميل التقرير"

## ⚙️ الإعدادات المتقدمة

### تغيير الألوان:
عدّل ملف `.streamlit/config.toml`:
```toml
[theme]
primaryColor="#1f77b4"     # اللون الأساسي
backgroundColor="#ffffff"   # لون الخلفية
```

### زيادة حجم الرفع:
```toml
[server]
maxUploadSize=200  # بالميجابايت
```

## 🐛 حل المشاكل
